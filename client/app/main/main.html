DatePicker Demo

Native (HTML5)

  • Supports date range restrictions.
  • Otherwise pretty limited in terms of restricting users.
  • Best mobile device integration because it uses native device controls.

Using <datalist> which is a white list of allowed helper list of dates:

Borrowed from http://tiffanybbrown.com/2013/10/24/date-input-in-html5-restricting-dates-and-thought-for-working-around-limitations/

Bootstrap (ui.bootstrap)

  • Supports min/max dates.
  • Can disable specific days of the week (e.g. weekends).
  • Not sure what's going on with the inital format of the date in the input, but I'm sure that can be resolved.

angular-datepicker

As a button:

Select a date: {{(selectedDate|date)}}

As an input:

  • Light-weight.
  • Doesn't support disabling individual dates out-of-the-box.
  • Doesn't support disabling weekends out-of-the-box.
  • The button option doesn't auto-close (out-of-the-box) because it's based on bootstrap dropdown-menu. But a simple modification resolves this.

ngQuickDate

  • Light-weight. No additional dependencies beside Angular.
  • Can disable specific days of the week (e.g. weekends).
  • Can disable specific dates.

angular-ui ui-date

  • This is an angular wrapper around jQueryUI's DatePicker. All options that are available for that will work. jQueryUI is required.
